Get timely push notifications about severe weather. Our Global Weather Alerts product allows you to set up weather alert push notifications in your projects and apps.

We collect weather warnings from majority of national weather warning systems and present them in a uniform and useful view.

Features of Global Weather Alerts

  1. We provide all weather alerts from the entire world.

    We collect weather alert notifications from majority of national weather warning systems worldwide. As soon as we get a new weather alert, we process it and send to our users via push notifications. Users get all weather alert notifications sent to their specified endpoint. We send notifications about new weather alerts and 15 minutes before they start, alert updates and alert cancellations.

  2. All necessary data in the push notification.

    Each push notification contains the necessary fields, such as the headline of the alert, its start and end dates, and a description. Additionally, each alert has data about urgency, severity, certainty and sender agency, to help users classify alerts according to various criteria. Here is a full list of the fields in the alert push notification.

  3. Monthly subscription.

    Please refere to the pricing page.

How to get access to Global Weather Alerts

There are two simple steps:

  1. To obtain the access to our Global Weather Alerts, please contact us at info@openweathermap.org.

    Please specify the endpoint that all weather alerts should be sent to.

    Example of the endpoint

    http://test_url.test
  2. Get all weather alerts to the specified endpoint via POST requests.

Fields in alert notifications

  • alert
    • id Alert ID
    • geometry GeoJSON representation of the area, where the alert is forecasted
  • msg_type Notification type. Please see the full list of notifications types here
  • categories Category of the alert. May be one of Geo, Met, Safety, Security, Rescue, Fire, Health, Env, Transport, Infra, CBRNE, Other
  • urgency Urgency of the alert. May be one of Immediate, Expected, Future, Unknown
  • severity Severity of the alert. May be one of Extreme, Severe, Moderate, Minor, Unknown
  • certainty Certainty of the alert. May be one of Observed, Likely, Possible, Unlikely, Unknown
  • start Date and time of the start of the alert, Unix, UTC
  • end Date and time of the end of the alert, Unix, UTC
  • sender Name of the alert source. Please read here the full list of alert sources
  • description
    • description.language Language of the alert's event name, headline, description and instruction. Please note that this field can be empty
    • description.event Alert evente name
    • description.headline Headline of the alert
    • description.description Description of the alert
    • description.instruction Instructions from agency about the alert
Weather alerts are provided in English by default.
Please note:
  • Some agencies provide the alert’s event name, headline, description and instruction only in a local language.
  • Some agencies provide the alert’s event name, headline, description and instruction in several languages. In this case, all available languages will be listed in the "description" section.

Types of alert notifications

cancel Notification of alert cancellation
